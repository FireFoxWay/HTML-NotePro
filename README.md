# Memoria NotePro

A lightweight, futuristic notes web app built with **HTML + CSS + JavaScript** (no frameworks).
It runs entirely in the browser and stores notes locally for fast access.

## Features
- Study notes with subjects
- Extra notes (permanent)
- Search + subject filtering
- Edit / delete notes
- Command console (help, time, notes, extra, search, subject, export, import)
- Export / import JSON backup

## Live Website
If hosted using GitHub Pages, your site will be available at:

- `https://<your-github-username>.github.io/<repo-name>/`

## How to Run Locally
1. Download or clone this repository
2. Open `index.html` in your browser

## Data Storage
Notes are stored using your browser storage (**localStorage**).  
This means notes are saved on the same device/browser you used.
